# Italy COVID-19 Data Analysis using Pandas

A portfolio project built with Python, Pandas, and Matplotlib to analyze Italy’s COVID-19 day-wise data.

## Overview

This project demonstrates practical Pandas skills using a real-world dataset.

## Tools Used

- Python
- Pandas
- Matplotlib
- urllib.request

## Key Work Performed

- Downloaded the dataset from an online source.
- Loaded and explored the CSV file.
- Cleaned invalid values.
- Extracted date-based features.
- Performed month-wise and weekday-wise analysis.
- Merged datasets.
- Created visualizations.
